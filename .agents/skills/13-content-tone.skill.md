# Instrucciones de la Skill

## Tono de la app

La app debe sonar como un coach cercano, no como médico ni militar.

El tono debe ser:

- Claro.
- Motivador.
- Práctico.
- Honesto.
- Sin juicio.
- Cero alarmista.

## Evitar

No usar frases como:

- “Estás fallando.”
- “Comiste mal.”
- “Entrenaste horrible.”
- “Debes bajar de peso.”
- “Si no haces esto, no progresas.”
- “Esta dieta te hará bajar X kilos.”

## Preferir

Usar frases como:

- “Hoy tu recuperación parece baja.”
- “Conviene priorizar descanso.”
- “Buen esfuerzo, pero no fuerces otro día intenso seguido.”
- “Tu constancia va mejorando.”
- “Este entrenamiento tuvo carga alta.”
- “Puedes enfocarte en hidratación y sueño hoy.”

## Mensajes de éxito

Ejemplos:

```txt
Entrenamiento guardado.
Buen trabajo. Ya tienes una nueva entrada para tu progreso.
```

```txt
Check-in registrado.
Con estos datos podemos ajustar mejor las recomendaciones de hoy.
```

## Mensajes de advertencia

Ejemplos:

```txt
Tu recuperación parece baja hoy. Considera bajar intensidad o hacer movilidad.
```

```txt
Dormiste poco y reportaste dolor alto. No parece el mejor día para buscar un PR.
```

## Empty states

Ejemplos:

```txt
Aún no tienes entrenamientos registrados.
Registra tu primer entrenamiento para empezar a ver progreso.
```

```txt
Todavía no hay suficientes datos para detectar avances.
Sigue registrando tus sesiones esta semana.
```
